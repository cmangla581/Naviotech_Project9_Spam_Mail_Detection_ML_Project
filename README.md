# Naviotech_Project9_Spam_Mail_Detection_ML_Project

This project consist of the Basic Spam Mail Detection Project which detects whether a mail or message is spam or ham, by predicting based on the dataset.  
Here, we have used the Naive Bayes ML Algorithm which is useful in predicting the text based result and also mainly used for NLP. 

# Accuracy of Model : 97.94% 
Here, I've also plotted the Confusion Matrix and Correlation Heatmap for the better analysis. 

Here, a streamlit predictor has been made for making it useful for the users.
